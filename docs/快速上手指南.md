# 快速上手指南（2天完成版）

## 今天必须完成的事（第1天）

### 上午：环境准备
- [ ] 安装 Node.js（https://nodejs.org/）
- [ ] 安装 Python 3.8+（如果没有的话）
- [ ] 注册 DeepSeek 账号并获取API密钥
  - 地址：https://platform.deepseek.com/
  - 新用户有免费额度

### 下午：后端搭建
```bash
# 1. 进入后端目录
cd backend

# 2. 安装依赖
pip install flask flask-cors python-dotenv openai

# 3. 创建配置文件
# 复制下面的内容保存为 .env 文件
```

`.env` 文件内容：
```
DEEPSEEK_API_KEY=你从DeepSeek获取的密钥
```

```bash
# 4. 启动后端测试
python app.py

# 看到 "Running on http://0.0.0.0:5000" 就成功了
```

### 晚上：前端搭建
```bash
# 1. 进入前端目录
cd frontend

# 2. 安装依赖
npm install

# 3. 启动前端
npm run dev

# 访问 http://localhost:3000 查看效果
```

## 遇到问题怎么办？

### 后端启动失败
```bash
# 检查 Python 版本
python --version  # 需要 3.8+

# 尝试单独安装依赖
pip install flask
pip install flask-cors
pip install openai
```

### 前端启动失败
```bash
# 清除缓存重装
rm -rf node_modules
npm install
```

### API 调用失败
- 检查 .env 文件是否在 backend 目录下
- 检查 API 密钥是否正确
- 检查 DeepSeek 账号是否有余额

## 紧急备用方案

如果 DeepSeek API 用不了，可以换成：

### 方案1：使用智谱AI
1. 注册 https://open.bigmodel.cn/
2. 修改后端代码：
```python
client = OpenAI(
    api_key=os.getenv("ZHIPU_API_KEY"),
    base_url="https://open.bigmodel.cn/api/paas/v4"
)
```

### 方案2：本地模拟
如果连API都调不通，可以返回固定文本应付演示：
```python
@app.route('/api/translate', methods=['POST'])
def translate():
    return jsonify({
        "success": True,
        "translation": "这是一个模拟的翻译结果。"
    })
```
